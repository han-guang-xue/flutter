## TCP抓包名词解释与逻辑图

SYN: 表示建立连接，
FIN: 表示关闭连接，
PSH: 表示有 DATA数据传输，

RST表示连接重置。
Seq: 序号
Ack: 应答序号, 表示响应
Win: 接收窗口   
len: 发送数据包的长度
